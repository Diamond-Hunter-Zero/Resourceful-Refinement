#### **Description**
The Plunger is a throwable weapon which inherits from the trident. Like a trident, it can be held and thrown at entities or blocks from range. It only deals 3 points of damage on impact.

However, the plunger has an additional functionality; When held, players can use it to right click on fluid-containing blocks, and empty their contents.

**ID:** *plunger*


#### **Behaviour**
As a weapon, the Plunger behaves the same as a trident; It can be held and used to attack, or thrown and picked up again. It uses a different set of sound effects to the vanilla trident.

The plunger functionality is triggered by holding the plunger in the player's main hand, and right-clicking on any block with fluid tanks to empty all fluid tanks stored on that block. This includes existing Create content, and blocks added by this mod.

#### **Rendering**
The Plunger uses a Java item model for in-game rendering, and GUI.
